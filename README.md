ADB
===

Projeto do site de transcrição de Palestras.
O sistema consiste em um serviço PHP que administra as transcrições de arquivos de áudio.
A arquitetura do sistema é composta por:
-Login: Controle de contas e divisão dos trabalhos
-Transcrição: Onde se ouve o áudio e onde se registra o texto de transcrição
-Administração: Onde o administrador do sistema registra novos arquivos a serem transcritos e acessa informações e documentos referentes ao trabalho
